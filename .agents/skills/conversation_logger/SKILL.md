# 自動對話紀錄器 (Conversation Logger)

這個 SKILL 的目標是自動將使用者與助理之間的對話紀錄、重要決策或執行過程妥善保存。

## 指南與步驟

當你需要記錄對話時，請遵循以下步驟：

1. **確認日誌檔案位置**
   - 預設紀錄檔案的位置為專案目錄下的 `logs/conversation_history.md`。
   - 如果 `logs` 目錄不存在，請先建立它；如果不包含該檔案，請先建立它，並在檔案開頭加上 `# 對話與執行紀錄`。

2. **新增紀錄格式**
   - 每次新增紀錄時，請附加上當前的日期與時間。
   - 紀錄內容應包含使用者的請求（User Request）以及助理的處理結果或回覆。
   - 格式範例如下：

   ```markdown
   ## [YYYY-MM-DD HH:MM:SS]
   - **User:** [使用者提出的原始需求或問題]
   - **Assistant:** [執行的任務摘要、決策過程或給予的回覆]

   ---
   ```

3. **觸發時機**
   - 當使用者要求將對話或任務記錄下來時。
   - 完成了某些需要被總結的複雜任務或架構討論後。
   - 請自主摘要並使用檔案編輯工具將內容寫入（append）日誌檔案。
